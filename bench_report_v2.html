Amdahl's Law — Scaling Analysis

Frame time decomposes into a fixed serial cost (scene rendering) plus a per-drone cost that scales linearly: T(N) = Tserial + N × Tper_drone. We measure this by running 1 to 64 drones with identical configs.

Baseline (Trail Off) Decomposition

T(1) = 0.911ms    T(64) = 1.659ms

Tper_drone = (1.659 − 0.911) / 63 = 0.012 ms/drone

Tserial = 0.911 − 0.012 = 0.899 ms

Serial fraction at 64 drones: 0.899 / 1.659 = 54.2%


Trail On Decomposition

T(1) = 0.930ms    T(64) = 2.434ms

Tper_drone = (2.434 − 0.930) / 63 = 0.024 ms/drone

Tserial = 0.930 − 0.024 = 0.906 ms

Serial fraction at 64 drones: 0.906 / 2.434 = 37.2%

Trails double the per-drone cost (0.012 → 0.024 ms) while serial overhead stays constant — textbook parallel scaling.
